About Planetarium

Planetarium is a web application that stores and retrieves planet information from a database. Users may search for planet information or add their own planets to the mix!

Backed By AWS

The website itself is stored on an Amazon s3 bucket and deployed through a virtual http server.  The database utilizes the DynamoDB service availible on AWS.

Highly Availible

Information is provided fast and efficiently using a Content Delivery Network service avilible on AWS called Cloudfront. The website is also backed by an AWS Elastic Load Balancer and Auto Scaling.

Secure

HTTP server is protected within a private network and secured behind a bastion.
